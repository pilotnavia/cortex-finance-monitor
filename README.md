# Cortex Finance Monitor

Real-time markets and trading dashboard tracking global stock exchanges, central banks, commodities, forex, crypto, and economic indicators worldwide.

This project builds the **finance** variant by default.

## Quick start

```bash
npm ci
npm run dev     # development server (finance variant)
npm run build   # production build → dist/
```
